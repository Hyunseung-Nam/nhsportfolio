  • 실험 장비 직접 연결 → 이상 감지 → 스마트폰 즉시 알림
  • 시계열 데이터 저장 · 실시간 대시보드 · 장비 상태 모니터링
  • 설치 한 번으로 전체 스택 실행

React · FastAPI · TimescaleDB · Python · pyserial · Telegram Bot API · Docker

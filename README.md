## Files
- `index.html`: 면접관 중심 실무형 포트폴리오 콘텐츠
- `styles.css`: 레이아웃 및 UI 스타일
- `assets/`: 프로젝트 스크린샷 파일
- `vercel.json`: 정적 배포 설정

## Deploy to Vercel
1. Vercel에서 `Project` 폴더를 루트로 연결
2. Framework Preset: `Other`
3. Build Command: 비움
4. Output Directory: 비움 (정적 루트)
5. Deploy

## Local preview
정적 페이지라서 브라우저에서 `index.html` 파일을 직접 열어도 확인 가능.
